# 7. 前端设计系统方案

## 1. 整体设计

目标：
- 移动优先 + 响应式设计，样式风格与交互方式以移动端为准，桌面端基本可用
- 后续可扩展到微信小程序及 App

设计原则：
- 严格按照功能需求来设计页面，不要过度设计
- 保持页面简洁

## 2. V1 页面清单

- `/` 首页：赛事日程 + 世界排名前 20
- `/rankings` 排名页：支持按积分、胜率、交手次数排序
- `/players/[slug]` 球员详情：个人信息、头像、统计数据、比赛记录
- `/events` 赛事列表：展示历年赛事日程
- `/events/[eventId]` 赛事详情：比赛对战图、比赛结果

V2 预留：
- `/search` 搜索结果页（基于 LLM API 的自然语言搜索）
