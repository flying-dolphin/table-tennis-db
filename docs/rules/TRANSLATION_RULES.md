# 赛事名称翻译规范

## 核心规则

**年份必须放在翻译结果的开头**，格式为 `YYYY年`。

## 格式模板

### WTT 赛事
```
[年份] + [赛事级别] + [赛事类型] + [地点] + [序号](可选)
```

### ITTF 和 其他非WTT 赛事
```
[年份] + [赛事级别] + [地点] + [赛事类型] 
```

## 具体规范

### 1. 年份位置
- ✅ 正确：`2026年新加坡大满贯赛`
- ❌ 错误：`新加坡大满贯赛 2026`

### 2. 赛事级别前缀
| 英文 | 中文 | 说明 |
|------|------|------|
| WTT | WTT | 保持不变 |
| ITTF | ITTF | 保持不变 |
| ETTU | ETTU | 保持不变 |
| ITTF-ATTU | 国际乒联-亚乒联 | 简称 |
| ITTF-Oceania | 国际乒联大洋洲 | - |
| ITTF-Americas | 国际乒联美洲 | - |
| ITTF-Africa | 国际乒联非洲 | - |

### 3. 赛事类型
| 英文 | 中文 |
|------|------|
| Smash / World Smash | 大满贯赛 |
| Champions / Champions | 冠军赛 |
| Star Contender | 球星挑战赛 |
| Youth Star Contender | 青少年球星挑战赛 |
| Contender | 挑战赛 |
| Youth Contender | 青少年挑战赛 |
| Feeder | 支线赛 |
| Finals | 总决赛 |
| Championships | 锦标赛 |
| Cup | 杯赛 |
| Masters | 大师赛 |
| World Championships | 世界锦标赛 |
| World Cup | 世界杯 |

### 4. 地点后缀
- WTT比赛所有城市地点名称后加"站"：`多哈站`、`北京站`、`曼谷站`
- 多城市赛事保留连接符：`爱知·名古屋`
- 运动会保留全称：`亚洲运动会`、`青年奥林匹克运动会`

### 5. 序号/届数
- 罗马数字紧跟地名后：`旧金山站II`、`伊斯坦布尔站II`
- 阿拉伯数字届数保留原词：`第二十五届中美洲和加勒比地区运动会`

### 6. 特殊赛事
| 类型 | 翻译示例 |
|------|----------|
| Para（残障） | 世界残疾人乒乓球[类型]赛 |
| Youth | 青少年 |
| U21/U13 | U21/U13 |
| Team | 团体 |
| Mixed Team | 混合团体 |
| Individual | 个人 |
| World Tour Grand Finals | 世界巡回赛总决赛 |

## 注意事项

1. **不翻译**：WTT、ITTF、ETTU、ATTU 等组织名称
2. **不翻译**：赞助商名称（如 AITO、Galaxy Entertainment Group）
3. **不翻译**：人名地名（音译即可）
4. **地名优先**：中国城市使用中文，其他音译
5. **保持简洁**：避免过长翻译，移除冗余词汇
